# 修复依赖安装错误

## 错误原因

`@tarojs/redux@^3.6.0` 版本不存在。最新版本是 `2.2.10`。

而且项目已经不再使用 Redux（在 `src/app.tsx` 中已经移除了），所以可以删除这个依赖。

## ✅ 已修复

我已经：
1. 从 `package.json` 中移除了 `@tarojs/redux` 依赖
2. 删除了未使用的 store 文件

## 现在请重新安装

在 CMD 中运行：

```cmd
pnpm install
```

这次应该可以成功安装了！

## 安装完成后

安装成功后，运行：

```cmd
pnpm dev:weapp
```

开始编译项目。
