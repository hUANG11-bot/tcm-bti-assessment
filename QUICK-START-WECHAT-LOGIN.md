# 微信一键登录快速开始

## 🚀 5分钟快速配置

### 步骤 1：获取 AppSecret（1分钟）

1. 访问 https://mp.weixin.qq.com 并登录
2. 进入：**开发** → **开发管理** → **开发设置**
3. 找到 **AppSecret（小程序密钥）**
4. 如果未设置，点击"生成"并保存（只显示一次！）

### 步骤 2：配置环境变量（1分钟）

在项目根目录创建 `.env` 文件：

```env
WX_APPID=wx04a7af67c8f47620
WX_SECRET=你刚才获取的AppSecret
```

### 步骤 3：启动服务器（1分钟）

```bash
# 确保已安装依赖
pnpm install

# 启动服务器（会自动读取 .env 文件）
pnpm dev
```

### 步骤 4：启动小程序（1分钟）

```bash
# 新开一个终端窗口
pnpm dev:weapp
```

### 步骤 5：测试登录（1分钟）

1. 在微信开发者工具中打开 `dist` 目录
2. 点击"我的"标签
3. 点击"微信一键登录"按钮
4. 授权后即可登录成功！
