# 一步一步操作指南

## 🎯 目标：编译小程序项目

## 步骤1：找到项目文件夹

1. 打开"文件资源管理器"（桌面上的文件夹图标，或按 `Win + E`）
2. 在左侧找到"此电脑"或"我的电脑"
3. 点击 `D:` 盘
4. 找到 `tcm-bti-assessment` 文件夹
5. 双击进入这个文件夹

## 步骤2：在项目文件夹中打开命令行

### 方法A：最简单的方法（推荐）

1. 在 `tcm-bti-assessment` 文件夹的**空白处**（不要点击文件）
2. **按住键盘上的 Shift 键**
3. **同时右键点击**空白处
4. 会弹出一个菜单
5. 选择"**在此处打开 PowerShell 窗口**"或"**在此处打开命令窗口**"
6. 命令行窗口会自动打开，并且**已经在项目目录中**了！

### 方法B：使用地址栏

1. 在文件资源管理器的**地址栏**（顶部显示路径的地方）
2. 输入：`cmd` 或 `powershell`
3. 按回车
4. 命令行会在当前文件夹打开

## 步骤3：确认目录正确

命令行窗口打开后，应该看到类似这样的提示符：

```
d:\tcm-bti-assessment>
```

或者：

```
PS d:\tcm-bti-assessment>
```

如果显示的是 `C:\Users\Administrator>`，说明不在项目目录，需要重新操作。

## 步骤4：安装依赖

在命令行中输入（直接复制粘贴）：

```
pnpm install
```

按回车，等待安装完成（可能需要几分钟）。

## 步骤5：编译项目

安装完成后，输入：

```
pnpm dev:weapp
```

按回车，开始编译。

## 步骤6：等待编译完成

编译过程中会显示很多信息，最后应该看到：

```
✔ 编译成功
```

或者类似的成功提示。

## 步骤7：在微信开发者工具中打开

1. 打开微信开发者工具
2. 点击"导入项目"
3. **项目目录**选择：`d:\tcm-bti-assessment\dist`
   - ⚠️ 注意：是 `dist` 文件夹，不是项目根目录
4. AppID：`wx04a7af67c8f47620`
5. 点击"确定"

## 🆘 如果遇到问题

### 问题1：找不到"在此处打开 PowerShell"选项

**解决**：
- 确保按住 **Shift** 键
- 右键点击的是**空白处**，不是文件
- 或者使用方法B（在地址栏输入 cmd）

### 问题2：命令行显示错误的目录

**解决**：
- 关闭命令行窗口
- 重新按照步骤2操作
- 或者手动输入：`cd d:\tcm-bti-assessment`

### 问题3：pnpm 命令找不到

**解决**：
- 先安装 pnpm：`npm install -g pnpm`
- 或者使用 npm：`npm install` 然后 `npm run dev:weapp`
