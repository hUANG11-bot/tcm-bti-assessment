# 版本描述（审核用）

## 版本 1.0.0

### 功能描述

TCM-BTI 中医体质评估系统是一款基于传统中医理论的体质测评小程序。

**核心功能**：
1. **体质测评**：通过科学的四维评估体系，准确识别用户体质类型
2. **详细报告**：提供包含主要体质、兼有体质、可视化图表、详细说明的完整报告
3. **个性化建议**：根据体质类型提供饮食、运动、作息、情绪、穴位等全方位调理建议
4. **用户系统**：支持微信一键登录，自动保存测评记录
5. **管理后台**：提供数据统计、分析、导出等管理功能

**适用人群**：关注健康的中青年人群，希望了解自身体质并获得个性化健康建议的用户。

**使用流程**：填写基本信息 → 回答测评问题 → 查看详细报告 → 获取调理建议
