# TCM-BTI 中医体质评估系统 - 版本描述

## 版本 1.0.0

### 📱 产品简介

TCM-BTI 中医体质评估系统是一款基于传统中医理论的体质测评小程序，帮助用户了解自身体质类型，获得个性化的健康调理建议。

### ✨ 核心功能

#### 1. 体质测评
- **四维评估体系**：基于能量性质（阴/阳）、生命力强度（虚/实）、津液状态（燥/湿）、气血运行（滞/敏）四个维度进行综合评估
- **智能算法**：采用科学的评分算法，准确识别用户体质类型
- **传统体质映射**：将四维评估结果映射到传统中医九种体质类型（气虚质、阳虚质、阴虚质、痰湿质、湿热质、血瘀质、气郁质、特禀质、平和质）

#### 2. 详细报告
- **主要体质展示**：清晰展示用户的主要体质类型
- **兼有体质识别**：识别用户的次要体质特征
- **可视化图表**：柱状图展示九种体质的得分情况
- **体质变化趋势**：提示体质随年龄的变化规律
- **详细说明**：包含体质含义、形成原因、常见困扰、注意事项、日常建议等全方位解读

#### 3. 个性化建议
- **饮食调理**：推荐宜吃和忌吃食物，提供推荐食谱
- **运动建议**：根据体质类型推荐适合的运动方式
- **作息调理**：提供作息和生活方式建议
- **情绪调理**：指导情绪管理和心理调节
- **穴位按摩**：推荐相关穴位及按摩方法

#### 4. 用户系统
- **微信一键登录**：支持微信小程序快速登录，无需注册
- **测评记录保存**：自动保存测评结果，方便后续查看
- **用户信息管理**：管理个人资料和登录状态

#### 5. 管理后台
- **数据统计**：总测评数、今日新增、用户总数等关键指标
- **体质分布分析**：可视化展示各种体质类型的分布情况
- **性别分布统计**：分析男女用户的测评数据
- **数据导出**：支持导出 CSV 格式数据，方便进一步分析
- **管理员系统**：支持管理员账户创建和权限管理

### 🎨 界面特色

- **简洁美观**：采用现代化的 UI 设计，界面简洁美观
- **绿色主题**：使用绿色作为主色调，符合健康主题

### 📖 使用说明

1. **开始测评**：点击首页"开始测评"按钮
2. **填写信息**：输入年龄、性别、选择生活习惯
3. **回答问题**：根据实际情况回答测评问题
4. **查看结果**：获得详细的体质评估报告
5. **获取建议**：查看个性化的调理建议
6. **保存记录**：登录后可自动保存测评结果
